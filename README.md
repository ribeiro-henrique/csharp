# Básicos:

## 1 - dotnet new console -o < nome da pasta >

## 2 - Program.cs, é o arquivo principal, onde será executado nosso código

## 3 - Nome.csproj, é o arquivo de config do projeto: controle de versão, libs instaladas, config do servidor (simiar ao package.json)

## 4 - Pasta Obj, recebe as dependencias instaladas após rodar "dotnet restore", similar ao "npm install"

# Comandos:

## dotnet new
## dotnet build -> compila e verifica a existencia de erros
## dotnet run
## dotnet test
